# 03 Knowledge

知识管理部负责知识入库、清洗、标签、去重、索引和归档。

## 升级后的职责

知识管理部的核心使命是：把材料变成资产。

重点负责：

- 资料入口管理：处理 inbox、网页、课程、代码片段、面试资料。
- 知识条目标准化：保留来源、上下文、摘要、标签、状态。
- 索引与检索：维护主题索引和标签体系，保证以后找得到。
- 去重与归档：避免重复堆积，不为了整洁丢失细节。
- 共享记忆候选：筛选稳定事实、偏好、重要决策，建议同步到共享记忆。
- 与学习发展部协作：要求知识条目可学习、可练习、可用于求职或项目。

## 轻量子角色

- `Knowledge Indexer Agent`：Archivist 的子角色，负责标签、索引、重复条目合并建议。
