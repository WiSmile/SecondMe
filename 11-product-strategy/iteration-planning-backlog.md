# 迭代规划建议需求池

本需求池用于承接每日复盘、Vince 指令、协作摩擦和 AI 热点中出现的优化建议。

目标不是立刻把所有好想法做掉，而是让 SecondMe 学会把好想法延迟决策，而不是延迟遗忘。

## 字段说明

- `ID`：需求编号，方便引用
- `优先级`：`P0` / `P1` / `P2`
- `建议启动时机`：`立即` / `本周` / `本阶段稍后` / `下个阶段` / `条件满足后`
- `状态`：`待观察` / `待评估` / `待确认` / `已确认` / `已排期` / `暂缓`
