# 11 Product Strategy

产品战略部负责把每日对话、画像变化、SecondMe 自我优化点和 AI 热点，转成可管理的产品规划、需求池和迭代建议，并持续跟进确认后的优化落地。

## 核心职责

- 发现 SecondMe 的产品问题、协作摩擦和增长机会
- 维护“迭代规划建议需求池”，避免好想法丢失或一次性做重
- 给需求标注优先级、建议启动时机和触发条件
- 为已确认需求提供推荐方案、关联部门和跟进状态
- 在每日复盘中输出产品战略视角

## 不负责的事

- 不替代总经办作为统一入口和总协调中心
- 不替代行为研究部管理 Vince 画像主档
- 不替代 Agent Factory 管理 agent 身份、岗位和提示词演进
- 不替代 Harness 管理权限、分支和执行边界
- 不跳过 Vince 直接改动重要规则
