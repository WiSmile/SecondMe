# 每日复盘留痕

这里用于索引每日复盘的结果。

日报通知以单独日报线程为主；仓库中保留必要的运行记录、画像观察、自我迭代记录和重要变更记录。

## 留痕规则

- 每次日报执行后，应在 `logs/run-log.md` 追加运行记录。
- 如有低风险文档自动更新，应在 `logs/change-log.md` 记录变更。
- 如产生待确认决策，应在 `logs/decision-log.md` 记录审批项。
- 如需要保留完整日报正文，可在本目录新增 `YYYY-MM-DD.md`。

## 当前机制

- 执行时间：每天 22:30，Asia/Shanghai
- 通知方式：单独日报线程
- 更新边界：低风险文档可自动更新，重要内容需要 Vince 确认
