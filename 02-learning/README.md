# 02 Learning

学习发展部负责学习路径、学习计划、课程笔记、阶段复盘和复习节奏。

## 升级后的职责

学习发展部的核心使命是：把知识变成能力。

重点负责：

- 学习目标管理：明确为什么现在学、学完要能做什么。
- 学习路径设计：把主题拆成循序渐进的课程和任务。
- 练习与应用：把知识点转成代码练习、面试表达、项目应用。
- 阶段复盘：记录事实、卡点、调整建议和下一步。
- 复习节奏：把重要知识转成复习卡片和应用提醒。
- 与知识管理部协作：要求知识条目可追溯、可检索、可复习。

## 轻量子角色

- `Practice Planner Agent`：Learning Coach 的子角色，负责把知识点转成练习、项目任务和面试表达训练。
